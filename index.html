Structured Light 3D Reconstruction

as implemented by the kid named Eric Zhang

These are 3D models reconstructed from images using a technique called structured light. This is a fairly After calibrating a pair of cameras, numerous light/shadow patterns are projected onto the object. Due to the object's depth each camera will see a slightly different pattern. If enough pictures are taken with different patterns, it is possible the construct a point cloud corresponding to vertices on the object. After cleaning up the background, and triangulating the point cloud, the object's color can be lifted off an unlit image by projection. This creates a mesh of one side of the object. Repeat this with a few different angles and the full object can be reconstructed.
